SYSTEM REQUIREMENTS

TrueBudget has been designed to work on the following Operating Systems and Excel versions.

  • Windows 7
  • Windows 8
  • Windows 8.1
  • Windows 10

  • Excel 2010
  • Excel 2013
  • Excel 2016
  • Excel 2019
  • Excel 365
  • ​El Capitan (10.11)
  • Sierra (10.12)
  • High Sierra (10.13)
  • Mojave (10.14)
  • Catalina (10.15)
  • Big Sur (11.0)

  • Excel 2011 (v 14.7.7)
  • Excel 2016 (v 16.16.27)
  • Excel 2019 (v 16.40)
  • Excel 365 (v 16.40)​

It may work on earlier versions of both Operating Systems, but we have not been able to test it on those.​
